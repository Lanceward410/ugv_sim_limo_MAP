Navigating to /usr/share/gazebo-9/ in Ubuntu 18.04:

Using the File Manager (Graphical Way):

  1. Open the File Manager: Click on the file cabinet icon, usually located on the left side of the screen or in the dock.

  1. Access the File System: In the left sidebar of the File Manager, you'll see a list of locations. Click on "Other Locations" at the bottom.

  1. Go to Computer: In the main area of the window, you'll see an icon labeled "Computer." Click on it. This will take you to the root directory (/) of your file system.

  1. Navigate to the Folder: Now, you'll see a list of folders. Double-click on the usr folder. Inside usr, double-click on share. Inside share, scroll down and find the gazebo-9 folder and double-click on it.

  1. View and Modify: Inside the gazebo-9 folder, you'll find subfolders like models and worlds. You can navigate into these to view and modify the contents.
